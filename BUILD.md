# Building 
Java Development Kit and Apache Maven are needed to use Orekit. Begin by installing the .exe version of JDK and the .zip
of Apache Maven. Once both installations are complete, make sure to extract Apache Maven and both it and JDK in Program 
Files, see figure 2.1.

<img width="500" height="411" alt="github1 (1)" src="https://github.com/user-attachments/assets/85e30a2b-88e2-40bd-86b7-a83950079810" />
  
  > Java and Apache Maven in Program Files

Open “Edit the System Environment Variables” through the Windows Start menu and click environment variables.

<img width="395" height="449" alt="image" src="https://github.com/user-attachments/assets/3de577bf-88cf-412f-9398-82764a132b77" />

> Windows System Properties

In this window, make sure to add the location of the Java file with “\bin” under Path of System Variables. Then add 
`M2_HOME` and `MAVEN_HOME` to User Variables with the location of Apache Maven in value. Add the Apache Maven location with 
“\bin” into the Path value of User Variables.
